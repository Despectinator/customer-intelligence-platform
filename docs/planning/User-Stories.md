# User Stories

## Overview

The Customer Intelligence Platform is designed for small business owners who want to manage customer data and gain actionable insights through customer segmentation. The following user stories define the expected functionality of the system from the end user's perspective.

---

# Authentication

### US-01

**As a new user, I want to register an account so that I can securely access the platform.**

**Acceptance Criteria**
- User can register using email and password.
- Duplicate email addresses are not allowed.
- Successful registration redirects the user to the dashboard.

---

### US-02

**As a registered user, I want to log in securely so that I can access my projects.**

**Acceptance Criteria**
- User enters valid credentials.
- Invalid credentials display an error message.
- User is redirected to the dashboard after successful login.

---

### US-03

**As a logged-in user, I want to log out so that my account remains secure.**

---

# Project Management

### US-04

**As a business owner, I want to create multiple business projects so that I can manage customer data separately for each business.**

---

### US-05

**As a business owner, I want to edit project information so that my business details remain up to date.**

---

### US-06

**As a business owner, I want to delete a project so that unused projects can be removed.**

---

# Customer Management

### US-07

**As a business owner, I want to add new customers so that I can build my customer database.**

---

### US-08

**As a business owner, I want to edit customer information so that customer records remain accurate.**

---

### US-09

**As a business owner, I want to delete customers so that inactive or incorrect records can be removed.**

---

### US-10

**As a business owner, I want to search customers so that I can quickly find specific customer records.**

---

# Transaction Management

### US-11

**As a business owner, I want to record customer purchases so that buying behavior can be analyzed.**

---

### US-12

**As a business owner, I want to edit transactions so that incorrect purchase information can be corrected.**

---

### US-13

**As a business owner, I want to delete transactions so that duplicate or incorrect records can be removed.**

---

# Customer Analytics

### US-14

**As a business owner, I want the system to automatically calculate RFM values so that customer behavior can be analyzed without manual calculations.**

---

### US-15

**As a business owner, I want customers to be automatically grouped into meaningful segments so that I can understand different customer types.**

---

### US-16

**As a business owner, I want to view customer segment statistics so that I can make better business decisions.**

---

### US-17

**As a business owner, I want to receive recommendations for each customer segment so that I know which marketing strategy to apply.**

---

### US-18

**As a business owner, I want the dashboard to update automatically whenever customer or transaction data changes so that the displayed insights always reflect the latest information.**

---

# Reporting

### US-19

**As a business owner, I want to visualize customer analytics through charts and graphs so that business trends are easier to understand.**

---

### US-20

**As a business owner, I want to monitor revenue contribution by customer segment so that I can identify my most valuable customers.**
